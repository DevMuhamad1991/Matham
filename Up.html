پەیوەندی Supabase

URL و Anon Key لە Supabase Project Settings → API بدۆزەرەوە. ئەم زانیارییانە تەنها لەسەر ئەم وێبگەڕە هەڵدەگیرێن (localStorage).

ڕێنمایی سازکردن

1. لە Supabase پڕۆژەیەکی نوێ دروست بکە (بەخۆڕایی).
2. بچۆ SQL Editor و فایلی supabase_schema.sqlـی پێشکەشکراو جێبەجێ بکە.
3. URL و Anon Key لە Settings → API کۆپی بکە و لێرە دایبنێ.
4. دوای پەیوەندیکردن، خواردن و کەتەگۆرییەکانی سەرەتایی فایلی مێنووەکەت بە دەستی زیاد بکە یان داوام لێ بکە دانەیەکی import script بۆت دروست بکەم.
